# UWSM-Data-Analysis
This repository was created to analyze data provided by the United Way of Southern Maine


This script will take a name of a csv file that follows a similar format to the one provided. It will then produce several graphs from the provided data for analysis.

The script will run with the provided file. If you wish to run with a different dataset, use the syntax uwsm_data_analyzer.main(file_name) or change the file name in the bottom of the script.

The outputs of this script will be 
- A grouped bar chart of the biggest challenges split by ethnicity
- A grouped bar chart of the biggest challenges split by age group
- A grouped bar chart of the biggest challenges split by ethnicity for specifically ALICE members
- A grouped bar chart of the biggest challenges split by age group for specifically ALICE members
- A bar chart of the 8 biggest challenges faced among all respondents
- A bar chart of the 8 biggest challenges faced among ALICE respondents
- A bar chart of the 8 biggest challenges faced among non-ALICE respondents
- Number of respondents in Southern Maine
- Number of ALICE respondents in Southern Maine
- Number of Non-ALICE respondents in Southern Maine
- A bar chart of the age breakdown of ALICE respondents
- A bar chart of the ethnicity breakdown of ALICE respondents
- A bar chart of how ALICE respondents answered how they wanted to get involved
- A bar chart of how the hardest expenses for ALICE respondents
- A bar chart of what community all respondents felt they were a part of that was unheard
- A bar chart of what community ALICE respondents felt they were a part of that was unheard
